## Station for Mac

１つのアプリでまとめて Web サイトを表示できるアプリ。

Twitter や Instagram, Linkedin, Facebook, Slack, Github などのサイトを 1 つの画面に収めて表示する機能があります。

以前紹介した Stack よりも軽くて使いやすい。対応アプリは 610 以上。

[Station](https://getstation.com/)
